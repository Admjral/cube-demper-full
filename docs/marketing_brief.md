# Cube Demper — Маркетинговое ТЗ для создателя презентаций

> Этот документ содержит полное описание платформы Cube Demper: модули, тарифы, целевую аудиторию и рекомендации по слайдам. Предназначен для дизайнера/создателя презентаций.

---

## 1. О платформе

**Cube Demper** — автопилот для продавцов на Kaspi.kz.

**Что это:** SaaS-платформа для автоматизации продаж на крупнейшем маркетплейсе Казахстана. Заменяет ручную работу продавца: управление ценами, аналитику, коммуникацию с клиентами и юридические вопросы.

**Ключевая ценность:** Продавец подключает магазин — платформа работает 24/7 без его участия.

**Целевой рынок:** Kaspi Marketplace — более 30 000 активных продавцов в Казахстане.

**Сайт:** cube-demper.shop

## 2. Модули платформы (12 модулей)

### 2.1 Демпинг цен (Price Bot)

**Главный модуль платформы.**

Автоматически отслеживает цены конкурентов и корректирует вашу цену, чтобы товар всегда был в топе выдачи Kaspi.

- **Режим 24/7** — бот проверяет цены каждые 15 минут (приоритетные товары — каждые 3 минуты)
- **Стратегии:**
  - *Standard* — быть дешевле самого дешёвого конкурента на заданный шаг
  - *Stay Top N* — держаться в top-N позиции (например, top-3)
- **Мульти-город** — разные цены в разных городах Казахстана (Астана, Алматы, Караганда и т.д.)
- **Демпинг по доставке** — учитывает сроки доставки конкурентов (сегодня, завтра, 3 дня, 5 дней)
- **Защита маржи** — минимальная и максимальная цена, ниже/выше которых бот не уходит
- **Исключение своих магазинов** — не конкурировать с собственными точками продаж
- **Приоритетные товары** — ускоренная проверка для самых важных позиций (до 10 товаров)

**Для кого:** Любой продавец, который хочет автоматизировать ценообразование.

---

### 2.2 Аналитика продаж

Полная картина бизнеса в одном дашборде.

- **Выручка** — общая, по дням/неделям/месяцам, графики
- **Заказы** — количество, статусы, динамика
- **Средний чек** — отслеживание тренда
- **Топ товаров** — рейтинг по выручке и количеству продаж
- **Фильтры** — по магазину, периоду, категории
- **Экспорт** — данные для скачивания

**Лимиты по тарифу:** Standard — 500 товаров, Plus — 1 000, Ultra — безлимит.

---

### 2.3 Поиск ниш

Инструмент для поиска прибыльных товарных ниш на Kaspi.

- **Анализ категорий** — количество продавцов, средняя цена, объём продаж
- **Метрики конкуренции** — насколько сложно войти в нишу
- **Поиск по ключевым словам** — находите категории по запросу
- **Оценка потенциала** — объём рынка и прогноз доходности

**Доступен:** начиная с тарифа Plus.

---

### 2.4 Юнит-экономика

Калькулятор маржинальности для каждого товара.

- **Автоматический расчёт** — комиссия Kaspi, стоимость логистики, себестоимость
- **Учёт всех расходов** — закупка, доставка, хранение, комиссия маркетплейса
- **Парсинг данных** — автоматическое получение актуальных цен с Kaspi
- **Маржинальность** — чистая прибыль на единицу товара
- **ROI** — окупаемость инвестиций

**Доступен:** во всех тарифах.

---

### 2.5 Предзаказы

Управление товарами с отложенной поставкой.

- **Срок предзаказа** — от 1 до 30 дней
- **Автоматическая передача в Kaspi** — срок предзаказа отправляется при синхронизации цен
- **Настройка по пунктам выдачи** — разные сроки для разных PP

**Доступен:** начиная с тарифа Plus или как аддон (10 000₸/мес).

---

### 2.6 WhatsApp рассылки

Автоматические сообщения клиентам через WhatsApp.

- **Авто-рассылка** — сообщение при триггерном событии:
  - Новый заказ → "Спасибо за заказ!"
  - Заказ доставлен → "Оставьте отзыв"
  - После покупки → кросс-сейл предложение
- **Массовая рассылка** — отправка по базе контактов (Ultra)
- **Шаблоны** — 7 предустановленных шаблонов + свои
- **Редактор шаблонов** — визуальный конструктор сообщений

**Авто-рассылка:** начиная с Plus. **Массовая:** только Ultra.

---

### 2.7 ИИ Продажник (AI Salesman)

ИИ-ассистент для допродаж через WhatsApp.

- **Технология** — Google Gemini (ИИ последнего поколения)
- **Автоматические сообщения** — предлагает сопутствующие товары после покупки
- **Персонализация** — анализирует историю покупок клиента
- **Настройка тона** — деловой, дружеский, нейтральный
- **Статистика** — конверсия, количество отправленных сообщений, ответы

**Доступен:** как аддон (15 000₸/мес) к любому тарифу.

---

### 2.8 Управление заказами

Все заказы из Kaspi в одном месте.

- **Синхронизация** — автоматическое получение заказов из Kaspi (каждые 8 минут)
- **Статусы** — новые, в доставке, на выдаче, требуют подпись
- **Данные клиента** — имя, телефон (реальный, не маскированный)
- **Мультимагазин** — заказы со всех подключённых магазинов
- **Уведомления** — пуш при новом заказе

**Доступен:** во всех тарифах.

---

### 2.9 Склейка накладных

Объединение PDF-накладных на одном листе A4.

- **Экономия бумаги** — несколько накладных на одном листе
- **Загрузка PDF** — загрузите файлы, получите объединённый документ
- **Быстрая печать** — готово к отправке в принтер

**Доступен:** во всех тарифах.

---

### 2.10 Интеграции Kaspi

Подключение магазинов к единому кабинету.

- **Мульти-магазин** — подключайте несколько магазинов Kaspi к одному аккаунту
- **Авторизация через Kaspi MC** — безопасный вход через личный кабинет продавца
- **SMS-подтверждение** — двухфакторная верификация при подключении
- **Автообновление сессии** — автоматическая переавторизация при истечении
- **API-ключ Kaspi** — подключение REST API для расширенных данных (заказы, реальные телефоны)
- **Store Points** — привязка пунктов выдачи (PP) к городам

**Доступен:** во всех тарифах.

---

### 2.11 ИИ-Юрист (AI Lawyer)

Юридические консультации по законодательству Республики Казахстан.

- **Технология** — Google Gemini + RAG (Retrieval-Augmented Generation)
- **База знаний** — законы РК: о защите прав потребителей, Гражданский кодекс, Налоговый кодекс, Закон о торговле и другие
- **Чат-формат** — задавайте вопрос на русском, получайте ответ со ссылками на статьи
- **История** — все консультации сохраняются в чате

**Доступен:** во всех тарифах.

---

### 2.12 Техподдержка

Встроенный чат с техподдержкой.

- **24/7** — поддержка доступна круглосуточно
- **Чат в приложении** — не нужно выходить из платформы
- **Приоритет** — подписчики платных тарифов обслуживаются в первую очередь
- **Уведомления** — пуш-уведомления о новых ответах

**Доступен:** во всех тарифах.

## 4. Целевая аудитория

### Персона 1: Начинающий продавец

- **Магазины:** 1-2
- **Товары:** 10-50
- **Боль:** Не успевает следить за ценами конкурентов, теряет позиции в выдаче
- **Решение:** Price Bot автоматически держит его в топе, аналитика показывает что продаётся
- **Тариф:** Standard (21 990₸)

### Персона 2: Средний продавец

- **Магазины:** 2-5
- **Товары:** 50-200
- **Боль:** Рутина съедает время, нет возможности масштабироваться. Хочет продавать в нескольких городах
- **Решение:** Мульти-город демпинг, WhatsApp рассылки для повторных продаж, поиск ниш для расширения ассортимента
- **Тариф:** Plus (27 990₸)

### Персона 3: Крупный продавец

- **Магазины:** 5+
- **Товары:** 200+
- **Боль:** Управление большим количеством товаров и магазинов, коммуникация с клиентами, юридические вопросы
- **Решение:** Ultra + аддоны. Полная автоматизация: демпинг, массовые рассылки, ИИ-допродажи, юрист
- **Тариф:** Ultra (33 990₸) + аддоны

### Персона 4: Дропшиппер / Новичок

- **Магазины:** 0-1
- **Товары:** планирует начать
- **Боль:** Не знает какой товар выбрать, не понимает маржинальность, боится конкуренции
- **Решение:** Поиск ниш для выбора товара, юнит-экономика для расчёта маржи, ИИ-Юрист для юридических вопросов
- **Тариф:** Standard + пробный период 3 дня
